How to Create a UMD Waitlist Notifier

Using mostly just Python and the command line

Through this guide, I will show you how to code and asynchronously run a Python script that will text your phone if a spot opens up in a UMD class. Such a script would be useful for classes like the introductory Computer Science ones, as these often have capped waitlists, and once those waitlists run out there are open spots in the class to grab. This guide will involve the use of the Python programming language and ideally a Raspberry Pi to have the script run as long as it needs to in the background.

Tools List

There aren't many physical tools you need for this project, mainly just some easy to install software:

  • Twilio account
  • Python 3
  • Any command line application, like Terminal on macOS or Command Prompt on Windows
  • A text editor of your choosing, my favorites being Atom and Sublime Text

Parts List

The parts you need for this project are simple:

  • Computer
  • Phone (with number)
  • Raspberry Pi (this is the preferred method of running your script, but it is not necessary)

Instructions

So now we've reached the good stuff; it's time to put together this waitlist notifier for UMD classes. These instructions below will hopefully get the entire project put together for you, but if not, be sure to check out the Troubleshooting section below.

Double-check that Python 3 is installed

To check if Python 3 is installed, open your command line application and type python. If the command is not found, head over to python.org where you can download it.

Clone the script from my GitHub page

To download the script, head to GitHub or type git clone https://github.com/ loganoconnell/Waitlist-Notifier in your terminal application.

Enter the directory with the script

In your terminal application, use the commands
cd directory to go into a directory, just cd to go to the root directory, and ls to view the contents of a directory. Once the output of ls contains the file waitlist.py, you've entered the right place.

Sign up for Twilio account

Head over to twilio.com, where you can sign up for a free account. Once you have an account made, head to the next step.

Record your API keys and phone number from Twilio

Once you're signed into your Twilio account, make sure to click on the Dashboard page and record both your Account SID, Auth Token, and Trial Number.

Fill in your API keys in the script

Open the script you've downloaded in the text editor you chose earlier. There will be a line that reads client = Client("x", "y"): fill in both x and y, preserving the double quotes, with your Account SID and Auth Token.

Fill in your target phone number in the script

With the script still open, find the line that reads client.messages.create(to = "+123456 78912", from_ = "+12345678912", body = "A seat has opened up in " + class_code + "!"). Fill in your target phone number in the to field and your Trial Number in the from_ field. Also, feel free to alter the message in the body field; this will be the message you are texted if a spot opens up.

Change the target class code in the script

The final adjustment you will make to the script will be changing the class code to the one you wish to track. First, change the string in the line class_code = "CMSC351" to the UMD class code you wish to target. Next, go to the Testudo Schedule of Classes, and note the section number you wish to target inside the class' page. This section number is not the traditional section number, like 0101, but instead the actual number from the top the section is. For example, if there were three sections, 0101, 0201, and 0301, the section number you would note for 0301 would be 3, since it is third in the list. Then, change the integer in the line section_num = 2 to the section number you noted above.

Run the script

To run the script, you have to options. If you'd like to run the script on your own computer, you can type python3 waitlist.py in the terminal application you chose. If you are a little more adept and would like to go the Raspberry Pi route we established earlier, run nohup python3 waitlist.py & while logged in to your Raspberry Pi over SSH to have the script run silently in the background.

Wait

Sit back and relax, and hopefully you'll get a text soon that a spot is open for you!

Troubleshooting

Using mostly just Python and the command line

  • Make sure Python is installed correctly
  • Make sure your class code is accurate, follow comments in script
  • Make sure you typed in Twilio API key and phone number correctly
